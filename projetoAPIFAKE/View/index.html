Buscar todos os usuários


fetch('https://fakestoreapi.com/users')
  .then(res => res.json())
  .then(json => console.log(json))

Buscar um unico usuário

Entre com o ID para buscar o usuário


      fetch('https://fakestoreapi.com/users/{id}')
        .then(res => res.json())
        .then(json => console.log(json))
